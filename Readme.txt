◎実行方法
このディレクトリ内の全てのファイルをダウンロードした後、index.htmを実行するか、下記のURLを参照してください。

◎URL
https://web.p.cyber-u.ac.jp/~1901015407/part12/Index.html

◎概要と操作方法
3点先取の打ち合いブラウザゲームです。
「開始！」をクリックすると動作が始まります。
カーソルを青の星に当てて動かすと青の星が左右に動き、クリックで弾を打ちます。
上または下に付いている数字は残り点数で、弾が当たると1ずつ減ります。
残り点数が先に0になったほうの負けです。

◎演出ポイント
弾をオレンジの六角形にして、星が隕石を撃ち合って遊ぶイメージを演出しています。
